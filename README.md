# Speech Emotion Recognition

## Overview

Deep learning project for classifying human emotions from speech signals using neural network architectures.

## Features

* Speech emotion classification
* MLP and CNN model comparison
* Feature extraction and normalization
* Confusion matrix and performance evaluation
* Accuracy up to 92.36%

## Technologies

* Python
* PyTorch
* Librosa
* NumPy
* Scikit-learn
* Matplotlib

## Results

* MLP Accuracy: 86.11%
* CNN Accuracy: 92.36%
